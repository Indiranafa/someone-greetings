# someone-greetings

A web app for sending warm wishes and celebrating special moments.

Website interaktif untuk merayakan tahun baru, berisi timeline perjalanan, surat masa depan, galeri, dan playlist.

## Struktur Clean Architecture

- src/components: Komponen UI
- src/pages: Halaman utama
- src/features: Business logic
- src/services: Data access
- src/styles: CSS
- src/utils: Helper functions

## Cara Pengembangan

1. Jalankan `npm install` untuk install dependensi
2. Jalankan `npm start` untuk development
